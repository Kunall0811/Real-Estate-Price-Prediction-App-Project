### 🏠 Real Estate Price Prediction Web App

---

🌐 Live Demo
👉 https://realestate-price-prediction-app.streamlit.app/

---

📌 Overview

This project is an end-to-end Machine Learning web application that predicts real estate property prices based on user inputs such as location, square footage, number of bedrooms (BHK), and bathrooms.
The application is built using a trained regression model and deployed using Streamlit, providing real-time predictions through an interactive and user-friendly interface.

---

🚀 Features

🔮 Real-time property price prediction

📍 Location-based estimation

📐 Takes input like area (sq ft), BHK, bathrooms

🎨 Clean and responsive UI using Streamlit

🌐 Fully deployed and accessible online
